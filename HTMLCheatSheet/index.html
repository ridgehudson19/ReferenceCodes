Inline Vs. Block level Elements

Inline

  • Does not start on new line
  • Takes only necessary width
  • <span>, <img>, <a>, <strong>, <em>

Block Level

  • Starts on new line
  • Takes full width
  • <div>, <h1>-<h6>, <p>, <form>

This paragraph is made from a Block Level Element Tag and the text effects is caused by Inline Element Tags . Other examples of Inline Element Tags are links opened in the same window or by opening a new tab
